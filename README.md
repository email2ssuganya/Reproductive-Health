Reproductive Health Factors Analysis

📌 Project Overview

This project focuses on analyzing various factors that influence reproductive health.
The aim is to identify patterns, correlations, and key indicators that impact reproductive well-being, helping to generate meaningful insights for awareness and decision-making.

📌 Objectives

Analyze reproductive health-related data
Identify key factors affecting reproductive health
Perform data cleaning and exploratory data analysis (EDA)
Visualize trends and patterns using dashboards
Generate actionable insights

📌 Data Processing

The dataset was cleaned using Python by handling missing and error values. In Power BI, the data was further prepared by ensuring correct data types and applying filters. The analysis was performed using aggregated values such as sum and average to derive meaningful insights.


📌 Key Insights

Weight-related issues and lack of prenatal care are the most significant pregnancy risk factors
Diabetes shows fluctuating trends and requires continuous monitoring
Medical and care-related factors contribute more to pregnancy risks than lifestyle factors
